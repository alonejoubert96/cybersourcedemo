## Live Demo

Once the app is running (`./gradlew bootRun`), open **[http://localhost:8080](http://localhost:8080)** to access the CyberShop storefront. Browse products, add them to your cart, and check out using any of the supported payment methods — all pre-filled with valid sandbox test data.

### Checkout Flow

1. **Product Catalog** (`/`) — Browse 6 tech products and add to cart
2. **Cart** (`/cart`) — Review items, choose delivery options, select a payment method
3. **Checkout Wizard** (`/checkout?method=...`) — Multi-step: Contact Details → Payment Details → Review & Confirm
4. **Order Confirmation** — Transaction details, order number, shipping info
