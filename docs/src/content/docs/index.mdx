## Live Demo

Run `./gradlew bootRun` and open **[http://localhost:8080](http://localhost:8080)**. The storefront has test products you can add to cart and check out with — forms are pre-filled with sandbox test data so you can submit straight away.

### Checkout Flow

1. **Product Catalog** (`/`) — add items to cart
2. **Cart** (`/cart`) — review items, pick a payment method
3. **Checkout** (`/checkout?method=...`) — contact → payment → review & confirm
4. **Confirmation** — transaction ID, order number, status
